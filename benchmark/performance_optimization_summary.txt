阶段1: 快速优化 (预计20分钟)
─────────────────────────────────────────────────────────────────────────
1. 修改 src/utils/util.cpp 中的 L2_distance 函数
   
   原代码:
   ```cpp
   float L2_distance(const float* x, const float* y, int d) {
       float dist = 0.0f;
       for (int i = 0; i < d; i++) {
           float diff = x[i] - y[i];
           dist += diff * diff;
       }
       return dist;
   }
   ```

   优化代码:
   ```cpp
   #include <faiss/utils/distances.h>
   
   float L2_distance(const float* x, const float* y, int d) {
       float dist = 0.0f;
       faiss::fvec_L2sqr(&dist, x, y, d);
       return dist;
   }
   ```

2. 减少训练样本
   修改 src/core/distributed_index_ivf.cpp (line 85):
   ```cpp
   const int64_t n_train = std::min(
       static_cast<int64_t>(clustering_->k) * 30,  // 从64改为30
       num_vectors
   );
   ```

3. 重新编译
   ```bash
   cd build
   g++ -std=c++17 -O3 -fopenmp ../src/utils/util.cpp -o util.o
   ar rcs libdann_core_minimal.a *.o
   g++ -std=c++17 -O3 -fopenmp ../benchmark/dann_bench.cpp -o ../benchmark/dann_bench_opt \
       -I../include -I/usr/local/include -I/usr/include/hdf5/serial \
       libdann_core_minimal.a /usr/local/lib/libfaiss.a \
       /usr/lib/x86_64-linux-gnu/hdf5/serial/libhdf5_cpp.so \
       -L/usr/lib/x86_64-linux-gnu/hdf5/serial -lhdf5 \
       -Wl,--allow-multiple-definition \
       -L/usr/lib/x86_64-linux-gnu -lmkl_intel_lp64 -lmkl_intel_thread \
       -lmkl_core -liomp5 -lm -ldl -lpthread
   ```

⚠️ 注意事项
═══════════════════════════════════════════════════════════════════════════

1. 正确性验证
   - 确保优化后搜索引擎结果一致
   - 测试边界情况 (空查询, 重复向量等)

2. 浮点精度
   - FAISS和SIMD可能有微小差异
   - 设置合理的误差容忍度

3. 可移植性
   - AVX2需要x86_64架构
   - 确保目标机器支持优化选项

4. 调试
   - 向量化代码更难调试
   - 建议保留原始实现用于调试
